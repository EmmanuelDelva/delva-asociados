# Delva & Asociados

Sitio oficial — despacho jurídico boutique para la economía digital.

**Stack**: Next.js 15 · React 19 · Tailwind 3.4 · GSAP · Lenis · Fraunces + Inter Tight + JetBrains Mono.

## Desarrollo local

```bash
npm install
npm run dev
# http://localhost:3000
```

## Producción

```bash
npm run build
npm start
```

## Deploy a Vercel

1. Conectar el repo en https://vercel.com/new
2. Framework preset: Next.js (autodetectado)
3. Build & deploy automáticos en cada push a `main`

## Filosofía de diseño

Ver [PRODUCT.md](./PRODUCT.md) (propuesta de valor, audiencia, voz)
y [DESIGN.md](./DESIGN.md) (sistema de color, tipografía, motion).

Cuatro actos cromáticos: drenched-forest → editorial-bone → drenched-forest → editorial-bone. La transición ES la narrativa.
