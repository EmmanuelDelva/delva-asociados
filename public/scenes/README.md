## Si una imagen falta

El sitio no rompe. Las áreas sin imagen renderizan el `ProblemaScene` SVG animado de fallback. El hero sin imagen muestra solo el gradient + ampersand.

## Optimización

Recomendado:
- JPG calidad 80%
- Ancho máximo 1600px (Next/Image optimiza el resto)
- Tamaño objetivo: 200-400KB por imagen
